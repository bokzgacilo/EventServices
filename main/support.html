1. Log in: Visit the reservation system's website or application.
  2. Create an account: If you don’t have an account, you’ll need to create one. Provide necessary details like your name, email, and password.
  3. Log in: Use your credentials to log in to the system.
  4. Enter event details: Fill in the necessary information about your event, such as:
    • Event Name: The name of your event.
    • Date and Time: The start and end times of the event.
    • Location: The venue or online link if it’s a virtual event.
    • Description: A brief description of what the event is about.
    • Capacity: The maximum number of attendees allowed.

REMINDER THERE IS NO REFUND IF THE EVENT IS CANCELED!!
